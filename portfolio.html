Overview

Daybook turns a few daily inputs — mood, sleep, symptoms — into a plain-language read on what's changing, powered by LLaMA 3.3 through the Groq API. I built the app end-to-end, then designed and shipped its public-facing product page and storefront using AI-assisted, no-code workflows — going from idea to a live, purchasable product without writing the landing page by hand.

The product

A full-stack web app for logging daily health data and surfacing patterns in plain language.

  • • Built front-to-back with HTML, CSS, and JavaScript
  • • Integrated Groq's API with LLaMA 3.3-70B for real-time pattern analysis
  • • Deployed on Vercel with a GitHub CI/CD pipeline — zero hosting cost

The launch

A dedicated product page and storefront, designed and shipped without traditional coding.

  • • Directed AI-assisted design (Claude) for a responsive, accessible landing page
  • • Built light/dark mode, custom animations, and a live checkout flow
  • • Published and verified through the Gumroad CLI before going live
